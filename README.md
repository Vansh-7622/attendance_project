# Student Attendance Management System

## Overview
A simple web-based Student Attendance Management System built with PHP (PDO), MySQL, vanilla JavaScript (AJAX) and CSS.  
Features:
- Login (admin / teacher)
- Manage classes and students (add/list)
- Mark attendance (date-based, upsert)
- Attendance reports with CSV export
- Dashboard with quick stats

---

## Requirements
- XAMPP / WAMP / LAMP with PHP 8.x and MySQL
- Browser (Chrome/Firefox)
- Project directory (example used in this README): `C:\xampp\htdocs\attendance_project`
